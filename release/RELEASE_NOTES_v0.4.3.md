# FaceAce v0.4.3

这一版重点重做了文档导入体验和 Markdown 题目识别。

## 导入准确性

- 明确的 Markdown 题目标题按“一题一个处理单元”拆分，避免多道长题挤在一次 LLM 输出中被截断。
- 章节标题与题目内容隔离，避免下一章节标题混入上一题答案。
- 支持 `### 题1`、`### Q1`、`### 1.` 等常见 Markdown 题目标题。
- `.md/.txt` 增加 GB18030 编码兼容。

## 进度与错误处理

- 显示读取文档、处理分段、已识别题数和整体百分比。
- 单个分段失败时保留其他成功题目，并展示错误和原文预览。
- 失败任务或待审核任务支持重新处理。
- 处理中禁止删除任务，避免后台任务与文件清理冲突。

## Agent 辅助导入

- 页面直接内置可复制给 Agent 的处理指令。
- 内置简短 JSON 示例，不再引用发行版不存在的项目文档。
- 不再以完整题库导出文件作为格式示例。

## 其他

- 上传文件限制为 25 MB，并净化文件名以避免异常路径。
- 修复新增提示在深色模式下对比度不足的问题。
- 发行版 ZIP 仍不包含用户数据库、日志或 API Key。
