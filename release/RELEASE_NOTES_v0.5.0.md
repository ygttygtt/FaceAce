# FaceAce v0.5.0

这一版重点提升文档导入的稳健性、AI 批改的灵活度，以及长时间刷题时的连续体验。

## 智能文档导入

- 文档先解析为带原文位置的标题、段落、列表、表格和代码块，再判断题目边界。
- 小标题只作为候选证据，并区分题目、章节、答案、追问和不确定标题。
- 无明确题目标题时，使用自然内容块滑动窗口和重叠区域，由 LLM 返回原文题目范围。
- 明确题目标题强制一段一题；模型过度拆题时自动纠错，仍不符合时确定性合并。
- 新增题目数量、章节声明数量、遗漏、覆盖、重复与误拆审计。
- 审核页面展示明确标题数、最终题数、去重数量和边界风险。

## 更灵活的 AI 批改

- 题库参考答案与评分点改为辅助材料，认可语义等价、不同术语和其他正确解法。
- 参考材料存在错误或遗漏时，模型可以指出，而不是要求用户机械复述。
- 可选生成一份不依赖导入参考答案的 AI 独立解析，用于交叉验证。
- 每次批改后可继续追问，问答记录绑定本次作答并持久保存。

## 低分重练与作答对比

- 错题重练可按最近一次得分设置阈值，提供 40、50、60 分快捷选项。
- 答好后自动移出对应低分集合，旧作答和旧评分不会删除。
- 同一道题的历次答案、分数变化和完整批改可以并排查看。

## 刷题连续性

- 当前题目、整轮题目、逐题草稿、揭晓状态和批改结果自动保存在本地。
- 切换到题库、历史或设置页面后，返回刷题页可以从原处继续。
- 页面刷新后同样能够恢复；主动结束本轮前会提示是否清空草稿。
- 增加上一题、完成后返回检查，以及独立解析失败的非阻断提示。

## 数据与兼容性

- 现有题库、作答、批改、导入任务和 LLM 配置自动迁移并保留。
- 新增独立解析字段和练习追问记录表。
- 公开发行 ZIP 不包含用户数据库、日志、导入暂存或 API Key。
