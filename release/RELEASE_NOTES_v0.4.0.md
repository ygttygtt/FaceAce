# FaceAce v0.4.0

这一版重点完善了日常使用和发行体验。

## 新增

- 全新的 FaceAce 品牌 Logo，覆盖 EXE、系统托盘、页面导航与浏览器标签。
- LLM 配置支持检测 API Key / Base URL，并自动获取可用模型供下拉选择。
- DeepSeek、美团龙猫、商汤日日新快捷预设，默认 DeepSeek V4 Flash，并附申请入口。
- 界面自动跟随系统浅色或深色模式。
- Windows 便携发行版：解压后双击 `FaceAce.exe`，无需安装 Python 或 Node.js。

## 改进

- 优化刷题、仿真面试、导入审核和移动端布局体验。
- 仿真面试支持会话重试、只读回顾、面试官设定与更清晰的历史记录。
- 加强题目抽取、错题复习、分组练习和练习计时。

## 下载与使用

下载 `FaceAce-v0.4.0-win64.zip`，完整解压后运行 `FaceAce.exe`。程序数据保存在解压目录的 `data/` 文件夹中。
