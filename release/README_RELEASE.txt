FaceAce Windows 便携版
======================

使用方法
--------
1. 解压整个压缩包，不要只单独复制 FaceAce.exe。
2. 双击 FaceAce.exe。
3. 程序启动后会自动打开默认浏览器。
4. 右下角系统托盘的 FaceAce 图标可用于重新打开页面或退出程序。

首次使用
--------
进入“设置 → LLM 配置”，选择 DeepSeek、美团龙猫或商汤日日新预设，填写 API Key。

数据和备份
----------
- 数据库和导入文件保存在 data 文件夹。
- 备份时退出 FaceAce，然后复制整个 data 文件夹。
- 从开发版迁移数据时，可把 backend\data\faceace.db 复制为 data\faceace.db。
- 请勿公开包含个人 API Key 的数据库文件。

故障排查
--------
- 无法启动：确认 8000 和 5173 端口未被其他程序占用。
- 查看日志：logs\faceace.log。
- 页面没有自动打开：双击托盘图标，或访问 http://127.0.0.1:5173。

卸载
----
从托盘退出 FaceAce 后，删除整个文件夹即可。
